Specify the type of analysis you would like to run.
Full Analysis will run the cov-build, cov-analyze, and cov-commit-defects tools (in that order)
Incremental Analysis will run the cov-build, cov-run-desktop, and cov-commit-defects tools (in that order)

Specifically, the tools will be invoked with the following parameters:

cov-build --dir ${WORKSPACE}/idir build command
cov-analyze --dir ${WORKSPACE}/idir
cov-run-desktop --dir ${WORKSPACE}/idir --host ${COVERITY_HOST} --port ${COVERITY_PORT} --stream ${COV_STREAM} ${CHANGE_SET}
cov-commit-defects --dir ${WORKSPACE}/idir --host ${COVERITY_HOST} --port ${COVERITY_PORT} --stream ${COV_STREAM}

If the specified ${COVERITY_HOST} is an https address, --ssl will be automatically appended.
